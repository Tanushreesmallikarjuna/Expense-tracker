# 💰 Expense Tracker (Flask)

A simple web-based Expense Tracker built using Flask that allows users to manage their daily expenses efficiently.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 📊 Dashboard to view all expenses
- ➕ Add new expenses
- ✏️ Edit existing expenses
- ❌ Delete expenses
- 💾 Data stored using SQLite

---

## 🛠️ Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS
- Database: SQLite
